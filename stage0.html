The Basics of the web

The web-World Wide Web- is a collection of HTML documents. The web was invented in the 1990s and has about 20 billion pages

HTML stands for HyperText Markup Languagewhich glues everything together and is the main document type of the web

Links between pages are called hyperlinks or links for short. These make the web web-like

URL is how you refer to documents on the web

When your browser makes a request via the internet to servers, the requests are using a protocol called HTTP and the servers respond with files that the browser ends up displayiing.

Browser: A program that runs on your computer to display files found on the web
Internet: World's largest computer network
HTTP: The main protocol of the web
Servers: Computers that host the files that make up the web.
